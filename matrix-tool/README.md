# Matrix Operations Tool

A modern GUI application for performing matrix operations, built with Python and CustomTkinter.

## Features
-   **Modern UI:** Dark-themed, clean interface using CustomTkinter.
-   **Dynamic Inputs:** Automatically adjusts input grids based on dimensions.
-   **Supported Operations:**
    -   Addition
    -   Subtraction
    -   Multiplication
    -   Transpose
    -   Determinant

## Usage

1.  **Install Requirements:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Launch Application:**
    ```bash
    python main.py
    ```

3.  **How to Use:**
    -   Select an operation from the sidebar.
    -   Enter matrix dimensions (Rows/Cols).
    -   Click "Generate Matrices" to create input fields.
    -   Fill in the numbers.
    -   Click "Calculate Result".
